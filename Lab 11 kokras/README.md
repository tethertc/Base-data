# Задание: Обработка ошибок и исключений в Python

## Цель
Научиться обрабатывать ошибки и использовать отладку в PyCharm.

## Используемые конструкции
- try / except / finally
- raise
- Структурное программирование (функции, main)

## Как запустить
1. Открыть проект в PyCharm
2. Запустить `main.py` (Shift + F10)
3. Ввести числа и посмотреть результат
